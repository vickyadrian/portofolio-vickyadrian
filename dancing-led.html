Proyek: Implementasi Efek Pencahayaan Dinamis (Dancing LED) Berbasis IoT

Proyek ini mendemonstrasikan implementasi sistem pencahayaan dinamis yang dikenal sebagai "Dancing LED" menggunakan mikrokontroler Wemos D1 Mini (ESP8266). Tujuannya adalah untuk menciptakan pola nyala LED yang terprogram dan berulang, memberikan efek visual yang menarik. Proyek ini merupakan contoh fundamental dalam bidang Internet of Things (IoT) dan Sistem Tertanam (Embedded Systems), menunjukkan kemampuan mikrokontroler dalam mengendalikan aktuator secara real-time berdasarkan logika yang kompleks.

Visualisasi Proyek

Setup hardware proyek Dancing LED yang menunjukkan koneksi Wemos D1 Mini ke 8 LED melalui breadboard.

Spesifikasi Teknis

Komponen Deskripsi Fungsi dalam Proyek
Mikrokontroler Wemos D1 Mini (ESP8266) Unit pemroses utama yang menjalankan logika program dan mengendalikan output digital. Dipilih karena ukurannya yang ringkas, harga yang efisien, dan integrasi Wi-Fi built-in.
Aktuator 8x LED (Light Emitting Diode) Menampilkan efek visual "Dancing LED" melalui status nyala/mati yang terprogram.
Interkoneksi Breadboard & Kabel Jumper Media prototipe untuk menghubungkan komponen secara non-permanen dan memfasilitasi pengujian sirkuit.
Lingkungan Pengembangan Arduino IDE Digunakan untuk menulis, mengkompilasi, dan mengunggah kode (sketsa) ke mikrokontroler ESP8266.
Bahasa Pemrograman C/C++ (Arduino Sketch) Bahasa tingkat rendah yang digunakan untuk memprogram operasi hardware secara langsung.

Diagram Sirkuit dan Koneksi Hardware

Sistem ini dirancang untuk mengendalikan delapan (8) LED secara independen melalui pin General Purpose Input/Output (GPIO) pada Wemos D1 Mini.

Tabel Koneksi Pin Wemos D1 Mini ke LED:

LED ke- Pin Wemos D1 Mini Pin GPIO ESP8266
1 D1 5
2 D2 4
3 D3 0
4 D4 2
5 D5 15
6 D6 12
7 D7 13
8 D8 14

Catatan Penting: Kaki negatif (katoda) dari setiap LED harus dihubungkan ke pin Ground (GND) pada Wemos D1 Mini untuk melengkapi sirkuit.

Implementasi Kode (Arduino Sketch)

Logika program diimplementasikan dalam fungsi loop() untuk menciptakan enam (6) pola pencahayaan dinamis yang berbeda secara berulang. Setiap pola menggunakan fungsi digitalWrite() untuk mengatur status pin (HIGH/LOW) dan fungsi delay() untuk mengontrol durasi transisi antar status.

Deklarasi Pin

int b1 = 5;  // GPIO 5 (D1)
int b2 = 4;  // GPIO 4 (D2)
int b3 = 0;  // GPIO 0 (D3)
int b4 = 2;  // GPIO 2 (D4)
int b5 = 15; // GPIO 15 (D5)
int b6 = 12; // GPIO 12 (D6)
int b7 = 13; // GPIO 13 (D7)
int b8 = 14; // GPIO 14 (D8)

Fungsi Setup

Fungsi setup() menginisialisasi semua pin yang terhubung ke LED sebagai output.

void setup() {
    pinMode(b1, OUTPUT);
    pinMode(b2, OUTPUT);
    pinMode(b3, OUTPUT);
    pinMode(b4, OUTPUT);
    pinMode(b5, OUTPUT);
    pinMode(b6, OUTPUT);
    pinMode(b7, OUTPUT);
    pinMode(b8, OUTPUT);
}

Fungsi Loop (Pola Pencahayaan)

Fungsi loop() berisi logika utama untuk menghasilkan pola-pola LED. Pola-pola tersebut meliputi:

  1. Pola 1 (Menutup ke Tengah): LED luar menyala berpasangan menuju ke tengah.
  2. Pola 2 (Membuka ke Luar): LED tengah menyala berpasangan menuju ke luar.
  3. Pola 3 (Berjalan Maju): LED menyala satu per satu dari 1 ke 8.
  4. Pola 4 (Berjalan Mundur): LED menyala satu per satu dari 8 ke 1.
  5. Pola 5 (Pasangan Maju): LED menyala berpasangan dari kiri ke kanan.
  6. Pola 6 (Pasangan Mundur): LED menyala berpasangan dari kanan ke kiri.

Kode Lengkap Pola:

void loop() {
    // Pola 1: Menutup ke Tengah (LED 1-8, 2-7, 3-6, 4-5)
    digitalWrite(b1, HIGH); digitalWrite(b8, HIGH); delay(100);
    digitalWrite(b1, LOW); digitalWrite(b8, LOW);
    digitalWrite(b2, HIGH); digitalWrite(b7, HIGH); delay(100);
    digitalWrite(b2, LOW); digitalWrite(b7, LOW);
    digitalWrite(b3, HIGH); digitalWrite(b6, HIGH); delay(100);
    digitalWrite(b3, LOW); digitalWrite(b6, LOW);
    digitalWrite(b4, HIGH); digitalWrite(b5, HIGH); delay(100);
    digitalWrite(b4, LOW); digitalWrite(b5, LOW);
    
    // Pola 2: Membuka ke Luar (LED 4-5, 3-6, 2-7, 1-8)
    digitalWrite(b4, HIGH); digitalWrite(b5, HIGH); delay(100);
    digitalWrite(b4, LOW); digitalWrite(b5, LOW);
    digitalWrite(b3, HIGH); digitalWrite(b6, HIGH); delay(100);
    digitalWrite(b3, LOW); digitalWrite(b6, LOW);
    digitalWrite(b2, HIGH); digitalWrite(b7, HIGH); delay(100);
    digitalWrite(b2, LOW); digitalWrite(b7, LOW);
    digitalWrite(b1, HIGH); digitalWrite(b8, HIGH); delay(100);
    digitalWrite(b1, LOW); digitalWrite(b8, LOW);
    
    // Pola 3: Berjalan Maju (LED 1 -> 8)
    digitalWrite(b1, HIGH); delay(100);
    digitalWrite(b2, HIGH); delay(100);
    digitalWrite(b3, HIGH); delay(100);
    digitalWrite(b4, HIGH); delay(100);
    digitalWrite(b5, HIGH); delay(100);
    digitalWrite(b6, HIGH); delay(100);
    digitalWrite(b7, HIGH); delay(100);
    digitalWrite(b8, HIGH); delay(100);
    digitalWrite(b1, LOW); digitalWrite(b2, LOW); digitalWrite(b3, LOW); digitalWrite(b4, LOW);
    digitalWrite(b5, LOW); digitalWrite(b6, LOW); digitalWrite(b7, LOW); digitalWrite(b8, LOW);
    
    // Pola 4: Berjalan Mundur (LED 8 -> 1)
    digitalWrite(b8, HIGH); delay(100);
    digitalWrite(b7, HIGH); delay(100);
    digitalWrite(b6, HIGH); delay(100);
    digitalWrite(b5, HIGH); delay(100);
    digitalWrite(b4, HIGH); delay(100);
    digitalWrite(b3, HIGH); delay(100);
    digitalWrite(b2, HIGH); delay(100);
    digitalWrite(b1, HIGH); delay(100);
    digitalWrite(b1, LOW); digitalWrite(b2, LOW); digitalWrite(b3, LOW); digitalWrite(b4, LOW);
    digitalWrite(b5, LOW); digitalWrite(b6, LOW); digitalWrite(b7, LOW); digitalWrite(b8, LOW);
    
    // Pola 5: Pasangan Maju (LED 1-2, 3-4, 5-6, 7-8)
    digitalWrite(b1, HIGH); digitalWrite(b2, HIGH); delay(100);
    digitalWrite(b3, HIGH); digitalWrite(b4, HIGH); delay(100);
    digitalWrite(b5, HIGH); digitalWrite(b6, HIGH); delay(100);
    digitalWrite(b7, HIGH); digitalWrite(b8, HIGH); delay(100);
    digitalWrite(b8, LOW); digitalWrite(b7, LOW); digitalWrite(b6, LOW); digitalWrite(b5, LOW);
    digitalWrite(b4, LOW); digitalWrite(b3, LOW); digitalWrite(b2, LOW); digitalWrite(b1, LOW);
    
    // Pola 6: Pasangan Mundur (LED 8-7, 6-5, 4-3, 2-1)
    digitalWrite(b8, HIGH); digitalWrite(b7, HIGH); delay(100);
    digitalWrite(b6, HIGH); digitalWrite(b5, HIGH); delay(100);
    digitalWrite(b4, HIGH); digitalWrite(b3, HIGH); delay(100);
    digitalWrite(b2, HIGH); digitalWrite(b1, HIGH); delay(100);
    digitalWrite(b8, LOW); digitalWrite(b7, LOW); digitalWrite(b6, LOW); digitalWrite(b5, LOW);
    digitalWrite(b4, LOW); digitalWrite(b3, LOW); digitalWrite(b2, LOW); digitalWrite(b1, LOW);
}

Kesimpulan

Proyek Dancing LED ini berhasil diimplementasikan menggunakan Wemos D1 Mini, menunjukkan kemampuan dasar dalam pemrograman embedded system dan kontrol aktuator. Penggunaan platform ESP8266 membuka potensi untuk pengembangan lebih lanjut, seperti integrasi kontrol melalui Wi-Fi (misalnya, melalui antarmuka web atau aplikasi seluler) untuk mengubah pola atau kecepatan LED secara nirkabel. Proyek ini menjadi fondasi yang kuat untuk eksplorasi lebih lanjut dalam bidang IoT dan otomatisasi.
